Ways To Apply Ccs To Page

Applying External Ccs

Each HTML page must include a reference to the external style sheet file inside the element, inside the head section. External CSS is also written in a separate file

Applying Internal CCS

You need to include CSS rules within a style tag inside the HTML document's head.

Applying Inline CCS

Inline CSS is added directly inside an element's attribute and applies styles directly to HTML elements using the style attribute.
